# `@webhook/web`

Operator console and docs for Hikyaku (Vite + React + TypeScript).

Repo: https://github.com/0x4Nayan04/Hikyaku

```bash
# From repo root
pnpm --filter @webhook/web dev
```

App: http://localhost:5173 · Docs: http://localhost:5173/docs

API base URL comes from `VITE_API_URL` (see root `.env.example`).
